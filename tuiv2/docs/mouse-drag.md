# TUIv2 鼠标拖动功能

## 功能说明

TUIv2 现在支持通过鼠标拖动浮动窗口。

## 使用方法

### 移动浮动窗口

1. 点击浮动窗口的任意位置（除了右下角）
2. 按住鼠标左键并拖动
3. 释放鼠标左键完成移动

### 调整浮动窗口大小

1. 点击浮动窗口的右下角（resize handle，2x2 区域）
2. 按住鼠标左键并拖动
3. 释放鼠标左键完成调整

## 实现细节

### 鼠标事件处理

- `tea.MouseActionPress`: 检测点击位置，判断是否点击了浮动窗口
- `tea.MouseActionMotion`: 拖动过程中更新浮动窗口位置或大小
- `tea.MouseActionRelease`: 清除拖动状态

### 拖动模式

- `mouseDragNone`: 无拖动
- `mouseDragMove`: 移动模式
- `mouseDragResize`: 调整大小模式

### Z-order 管理

点击浮动窗口时，会自动将其提升到最顶层（Z-order 最高）。

## 测试

要测试鼠标拖动功能：

1. 启动 termx TUI：`./termx`
2. 创建一个浮动窗口（使用快捷键或命令）
3. 使用鼠标点击并拖动浮动窗口
4. 使用鼠标点击右下角并拖动来调整大小

## 限制

- 最小窗口宽度：10 字符
- 最小窗口高度：4 行
- 浮动窗口不能拖动到负坐标位置
