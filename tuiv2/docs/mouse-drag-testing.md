# TUIv2 浮动窗口鼠标拖动测试指南

## 如何测试鼠标拖动功能

### 1. 启动 termx
```bash
./termx
```

### 2. 创建浮动窗口
1. 按 `Ctrl+O` 进入 Floating 模式（状态栏会显示 "floating"）
2. 按 `n` 创建新的浮动窗口
3. 会弹出 terminal picker，选择一个终端或创建新终端

### 3. 测试鼠标拖动
- **移动窗口**：点击浮动窗口的任意位置（除了右下角），按住鼠标左键拖动
- **调整大小**：点击浮动窗口的右下角，按住鼠标左键拖动
- **聚焦窗口**：点击浮动窗口的任意位置

### 4. 验证功能
- 移动时窗口应该跟随鼠标移动
- 调整大小时窗口应该改变尺寸
- 点击窗口时应该激活该窗口（边框颜色可能会变化）

## 快捷键参考

### Normal 模式
- `Ctrl+O`: 进入 Floating 模式
- `Ctrl+P`: 进入 Pane 模式
- `Ctrl+T`: 进入 Tab 模式
- `Ctrl+F`: 打开 Terminal Picker

### Floating 模式
- `n`: 创建新浮动窗口
- `h/j/k/l`: 移动浮动窗口
- `H/J/K/L`: 调整浮动窗口大小
- `c`: 居中浮动窗口
- `r`: 提升浮动窗口到顶层
- `t`: 切换浮动窗口可见性
- `x`: 关闭浮动窗口
- `Esc`: 退出 Floating 模式

## 调试

如果鼠标拖动不工作，检查：

1. **是否有浮动窗口**：确保已经创建了浮动窗口
2. **鼠标模式是否启用**：termx 应该自动启用鼠标支持
3. **终端是否支持鼠标**：某些终端可能不支持鼠标事件
4. **是否在模态窗口中**：如果有 picker 或 prompt 打开，鼠标事件会被忽略

## 已知问题

- 如果浮动窗口没有绑定终端，可能无法正确渲染
- 某些终端模拟器可能不支持鼠标拖动
