# 场景 01：默认 Workbench

## 目标

定义用户执行 `termx` 后进入的默认工作台主画面。

## 状态前提

- 默认进入可工作 workspace
- 至少有一个 live pane
- 顶栏、pane 标题栏、底栏都可见
