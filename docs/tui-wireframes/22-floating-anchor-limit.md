# 场景 22：Floating 锚点极限位置

## 目标

定义 floating pane 被拖出主视口时的极限状态，明确“允许部分出界，但左上角锚点必须保留可见”。

## 状态前提

- 当前 tab 中存在 active floating pane
- 用户持续拖拽，尝试把 floating pane 移出可视区域

## 线框图

## 关键规则

- 浮窗允许大部分区域出界，以便给 tiled 区腾出视野
- 左上角拖动锚点必须始终留在主窗口内
- 这样用户才能可靠地重新拖回浮窗
- `center` 动作应总能把当前浮窗呼回到安全位置

## 流转

- drag outward -> stop at anchor limit
- center -> recall to safe center
- focus float -> auto raise to top
